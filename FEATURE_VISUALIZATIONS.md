## Design Principles

1. **Mobile-First**: All interfaces optimized for touch, with large tap targets
2. **Progressive Disclosure**: Show most important info first, details on demand
3. **Visual Hierarchy**: Use size, color, and spacing to guide attention
4. **Consistent Patterns**: Reuse components across different views
5. **Accessibility**: High contrast, clear labels, screen reader support

## Color Palette

- **Primary**: Blue (#3B82F6) - Actions, links
- **Success**: Green (#10B981) - Achievements, positive trends
- **Warning**: Amber (#F59E0B) - Attention needed
- **Danger**: Red (#EF4444) - Declining performance
- **Neutral**: Gray scale for backgrounds and secondary text
